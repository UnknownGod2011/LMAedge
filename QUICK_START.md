## üîß Troubleshooting

### "Analysis failed" Error
**Solution**: 
- Check internet connection
- Try a smaller PDF
- Ensure PDF contains text (not images)
- Refresh page and try again

### No Sections Appearing
**Solution**:
- Verify PDF is text-based
- Check if document is in English
- Try a different PDF
- Check browser console (F12) for errors

### Slow Processing
**Normal for**:
- Large PDFs (50+ pages)
- Complex documents
- First-time analysis

**Speed it up**:
- Use smaller PDFs
- Close other browser tabs
- Check internet speed

### Chatbot Not Responding
**Solution**:
- Ensure document is fully analyzed
- Check if sections are displayed
- Refresh page if needed
- Verify API key is configured
